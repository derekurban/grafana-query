# Security

`wabsignal` is a hosted Grafana CLI. Treat the following credentials as sensitive:

- Grafana HTTP API read tokens
- Grafana Cloud access-policy management tokens
- Project OTLP write tokens

## Storage model

- Global read and management tokens are stored in the OS keyring.
- Project write tokens are stored in the local config file by design.
- Normal CLI output redacts secrets. Usable values are only emitted by explicit token/bootstrap commands such as `wabsignal project env`.

## Release integrity

- Release artifacts are published by GitHub Actions.
- `checksums.txt` is generated by GoReleaser.
- The release workflow signs `checksums.txt` with Cosign using GitHub OIDC.

## Reporting

If you discover a security issue, do not open a public exploit report with live credentials or reproduction data that exposes secrets. Redact tokens, OTLP headers, and customer-identifying telemetry before sharing logs or traces.
